DEPLOYING ON VERCEL
===================
Push these files to the connected repo (or drag-drop into the Vercel project).
vercel.json handles clean URLs, the /pages/* routing, and the .well-known headers.
No build step required — this is a static site.

DEPLOYING ON A STANDARD (APACHE) SERVER
=======================================
Upload the whole folder to the web root. .htaccess reproduces the same routing.
Ensure mod_rewrite is enabled.
